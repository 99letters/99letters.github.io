## 判断 1 — スペック情報の表示

### 結論: 承認。ただし「メモ」として扱え

数値は必要だ。「思ったより小さかった」はラグジュアリー購買における最悪の体験だ。
しかし数値をUIの主役にするな。数値は「欲しいと思った後に確認するもの」だ。

**実装ルール:**

```
NG  スペック表（テーブル形式）
NG  ヒーローエリアへの配置
NG  商品名と同等のフォントサイズ
NG  ラベル付きの仕様書スタイル

OK  1行フォーマット: H 12  W 8  D 8 cm  /  280 g
OK  本文の60%以下のフォントサイズ
OK  カラーは #777 以下
```

写真が撮れるなら、手に持ったカット1枚の方が数値より伝わる。数値は補助だ。

## 判断 2 — ライフスタイル写真

### 結論: 条件付き承認。「生活感」を撮るな、「余白のある生活」を撮れ

生活感とは雑然とした食卓・こどもの落書き・使いかけの調味料だ。
それはこのブランドが翻訳しようとしている沈黙とは正反対のものだ。

**撮るべきもの:**
- 素材が際立つ背景。コンクリート、白い漆喰、黒い石、粗い木。
- 光は朝か夕方の一方向。影が長く、やわらかい。
- 人は映さない。ただし「使った後の痕跡」は許容する（テーブルに置かれた器、飲まれた後のカップ）。
- 1カットに工芸品は1点のみ。
- 食材・花・装飾品を添えない。それ自体で立っていなければ完成していない。

**撮ってはいけないもの:**
- 笑顔の人が器を持つカット
- テーブルに食事が並ぶカット
- インテリア雑誌的な「部屋全体」のカット
- ドライフラワー・キャンドルとの組み合わせ

参照: 杉本博司の静物写真。説明ではなく、存在だけがある。

## 判断 3 — ケアガイドのトーン

### 結論: 強く承認。このサイト最重要のコンテンツになり得る

ケアガイドは「取扱説明書」ではない。「関係の継続」を語るページだ。

工芸品を買う行為は所有の開始ではなく、関係の開始だ。
漆は使うほど艶が増す。信楽の土は手の脂を吸って色が変わる。
これは「劣化」ではなく「成長」だ。

**コピーの文法:**

```
NG  「食洗機は使用しないでください。」
NG  「乾いた布で拭き取ってください。」
NG  命令形全般

OK  「漆は、暗いところで乾く。あなたの手の熱が、器を変えていく。」
OK  「信楽の土は最初、あなたの手を拒む。何年か経つと、手の形を覚える。」
OK  「麻は水を嫌わない。洗うたびに、少しだけ柔らかくなる。」
```

**各素材の構造:**
1. **素材の性格** — この素材はどんな存在か（詩的な導入）
2. **時間との関係** — 使うほどどう変わるか
3. **付き合い方** — ケア方法（命令形不使用）
